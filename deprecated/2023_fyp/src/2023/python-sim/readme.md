# Running Simulation

This code runs a python simulation of the control system in 2 dimensions. The implementation was made during the validation and prototyping
period of the control system design. It is now mostly obsolete and superseded by gazebo simulations. Note, dependancies required to run 
the simulation are found in the requirements.txt file. 

## simulations.py

- To make another simulation, see the different formations which have been instantiated in simulations.py. Follow the conventions used. Change the formation you want   
  to simulate on line 11 of main.py
